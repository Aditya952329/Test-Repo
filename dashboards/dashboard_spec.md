# Dashboard Spec (Power BI / Tableau / Looker Studio)

## Page 1: Executive Overview
- KPI cards: Orders, Cancellation %, SLA Breach %, Revenue, Contribution Margin %
- Trend line: Daily orders vs cancellations
- Map: Store-level SLA breach in city clusters

## Page 2: Ops Deep Dive
- Heatmap: Category x Hour (Cancellation %)
- Scatter: Avg delivery mins vs cancellation rate by store
- Table: Worst 10 stores with drill-down

## Page 3: Profitability Lens
- Discount % vs margin (bucket analysis)
- Category margin waterfall
- Impact simulation: "What if SLA breach drops by 5%?"

## Filters
- Date range
- City
- Store
- Category
- Payment mode
